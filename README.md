# SprintBoard

SprintBoard is a collaborative project management application that helps teams create projects, manage tasks, assign work, and communicate through comments.

## Features

- User Authentication
- Create Projects
- Group Collaboration
- Task Management
- Task Assignment
- Comments
- Real-time Updates
- Firebase Firestore Database

## Technologies

- HTML
- CSS
- JavaScript
- Firebase Authentication
- Cloud Firestore

## How to Run

1. Clone the repository.
2. Open the project in VS Code.
3. Run with Live Server.
4. Register a new account.
5. Start creating projects.
